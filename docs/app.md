`mw app`
========

Manage apps, and app installations in your projects

* [`mw app versions [APP]`](#mw-app-versions-app)

## `mw app versions [APP]`

List supported Apps and Versions

```
USAGE
  $ mw app versions [APP]

ARGUMENTS
  APP  name of specific app to get versions for

DESCRIPTION
  List supported Apps and Versions
```
